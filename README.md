# TelecomX Challenge

## 📘 Descripción

**TelecomX Challenge** es una iniciativa destinada a analizar y visualizar datos del sector de telecomunicaciones en América Latina. Proporciona herramientas para procesar información, generar insights y presentar resultados detallados mediante notebooks interactivos.

## 🧩 Contenido del repositorio

- `TelecomX_Data.json` – Archivo con datos estructurados utilizados para análisis.
- `TelecomX_LATAM.ipynb` – Notebook en Google Collab con procesamiento de datos, generación de gráficos y conclusiones relevantes.

## 🚀 Instalación

Crea un nuevo notebook en Google Collab (Debes tener una cuenta personal creada)

Link: https://colab.research.google.com/


📊 Estructura del notebook
Carga de datos – Se importa TelecomX_Data.json mediante pandas.

Limpieza y transformación – Se depuran datos, se manejan valores faltantes y se crean variables relevantes.

Visualización – Se generan gráficos (líneas, barras, mapas de calor) que ilustran tendencias en el mercado de telecomunicaciones.

Conclusiones – Se destacan hallazgos clave sobre penetración de servicios, crecimiento por país y áreas de oportunidad.
